# Portafolio de Estructura de Datos

**Universidad Autónoma de Yucatán - Facultad de Matemáticas**
 by Paredes Vazquez Ian

---

## Listado de Actividades de Aprendizaje (ADAs)

### [ADA 01: Pilas y Expresiones](./ADA01)
Conversión de expresiones infijas a postfijas utilizando Pilas (Stacks).

### [ADA 02: Manejo de Memoria y Apuntadores](./ADA02)
Ejercicios prácticos en C sobre el manejo de memoria dinámica y estructuras base.

### [ADA 03: Ordenamiento y Archivos CSV](./ADA03)
Programa en C++ que lee una base de datos de películas (`Movies.csv`) para realizar búsquedas y ordenamientos.

### [ADA 04: Listas Enlazadas](./ADA04)
Implementación y manipulación de Listas Enlazadas (Linked Lists) para gestión dinámica de datos.

### [ADA 05: Investigación de Estructuras No Lineales](./ADA05)
Presentación e investigación documental sobre estructuras de datos avanzadas.

### [ADA 07: Árboles Binarios](./ADA07)
Implementación de Árboles Binarios de Búsqueda (BST) y recorridos.
